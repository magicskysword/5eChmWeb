工坊 Workshop
5级据点设施

先决条件：无
空间：宽敞
雇员：3
指令：制造

工坊是充满创意的地方，许多有用的东西都能在这里制造。

工匠工具 Artisans’s Tools ：工坊被加入据点时，其内有六种工匠工具Artisans’s Tools可以投入使用。从下述列表中进行选择：

  • 木工工具 Carpenter’s Tools
  • 鞋匠工具 Cobbler’s Tools
  • 玻璃匠工具 Glassblower’s Tools
  • 珠宝匠工具 Jeweler’s Tools
  • 皮革匠工具 Leatherworker’s Tools
  • 画家工具 Pianter’s Tools
  • 陶艺工具 Potter’s Tools
  • 石匠工具 Mason’s Tools
  • 修补工具 Tinker’s Tools
  • 织布工具 Weaver’s Tools
  • 木雕工具 Woodcarver’s Tools

制造选项 Craft Options ：当你对此设施下达制造指令时，从以下选项中选择一项：

  • 制造：冒险装备 Craft：Adventuring Gear ：设施雇员制造任意能用工坊内已有的工匠工具（见上文）制造的物品，使用玩家手册中的相关规则。
  • 制造：魔法物品（器具） Craft：Magic Item（Implement） ：如果你的等级达到或高于 9 级，你可以委托此设施的雇员制造一件由你选择的普通或非普通魔法物品，该物品必须来自第 7 章的器具 Implement表格。此设施拥有制造该物品所需的工具，而其雇员拥有该工具的熟练以及奥秘技能的熟练。参考第 7 章“制造魔法物品”一节来决定制造该物品所需的时间与金钱。如果该物品允许使用者用它施展任何法术，你必须亲自制造它（设施雇员可以协助你），并且在制造该物品期间，你每天都必须准备所有该物品可以施展的法术。

灵感之源 Source of Inspiration ：当你在工坊内完成一次完整的短休后，你获得英雄激励 Heroic Inspiration。你在完成一次长休前不能再次获得此增益。

扩大设施 Enlarging the Facility ：你可以花费 2000 GP ，将工坊扩大为庞大设施。这样做之后，工坊获得额外两位雇员以及额外三种工匠工具（从上述列表中选择）。
